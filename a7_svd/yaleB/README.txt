Human faces for eigenfaces examples:

allFaces.mat

Each column vector is a human face that has been reshaped from 192 x 168 (n x m) into a 32256x1 column vector.  There are faces from 38 individuals.  The number of faces for each individual are given in the vector nfaces.

This data is modified from the Extended Yale Face Database B.  If using this data, please also cite the following two papers:

[1] A. S. Georghiades, P. N. Belhumeur, and D. J. Kriegman, From few to many: Illumination cone models for face recognition under variable lighting and pose, IEEE Trans. Pattern Anal. Mach. Intell., 23 (2001), pp. 643

[2] K. Lee, J. Ho, and D. J. Kriegman, Acquiring linear subspaces for face recognition under variable lighting, IEEE Trans. Pattern Anal. Mach. Intell., 27 (2005), pp. 684

Link to website: vision.ucsd.edu/content/extended-yale-face-database-b-b
